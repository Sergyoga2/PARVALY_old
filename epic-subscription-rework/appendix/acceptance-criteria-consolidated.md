# Acceptance Criteria: Consolidated

## Phase A: Новые тарифы

### Тарифная сетка
- [ ] 4 тарифа отображаются на pricing page: 1/3/6/12 мес
- [ ] Цены корректны: 3 900 / 3 300 / 2 900 / 2 400 ₽/мес
- [ ] Итого: 3 900 / 9 900 / 17 400 / 28 800 ₽
- [ ] Скидки: — / 15% / 26% / 38%
- [ ] 6 мес помечен как «Выбор большинства»
- [ ] 3-летний тариф не отображается для новых пользователей
- [ ] Feature flag переключает между старой и новой страницей

### Покупка
- [ ] Неавторизованный → CTA → redirect на регистрацию → обратно
- [ ] Авторизованный без подписки → checkout → оплата → active
- [ ] Авторизованный с active подпиской → ошибка «уже есть подписка»
- [ ] Payment failed → ошибка, подписка НЕ создана
- [ ] Согласие с офертой обязательно (checkbox)

### Автопродление
- [ ] Monthly: списание каждый месяц по 3 900 ₽
- [ ] Quarterly: списание каждые 3 мес по 9 900 ₽
- [ ] Semi-annual: каждые 6 мес по 17 400 ₽
- [ ] Annual: каждые 12 мес по 28 800 ₽
- [ ] Reminder за 7 дней для 3/6/12 мес
- [ ] Неуспешное продление → уведомление

### Legacy
- [ ] Legacy monthly: renewal 3 900 ₽/мес без изменений
- [ ] Legacy annual: renewal 34 800 ₽/год без изменений
- [ ] Legacy 3-year: профессии доступны, conditions unchanged
- [ ] Legacy подписчик видит «архивный тариф» + баннер
- [ ] После отмены legacy → только новые тарифы

## Phase B: Free Trial

### Активация
- [ ] Новый пользователь (trial_used=false, нет подписки) может оформить trial
- [ ] Привязка карты обязательна
- [ ] Trial = 7 дней (168 часов от активации)
- [ ] Полный доступ к 56 навыкам, профессии заблокированы
- [ ] user.trial_used = true после активации
- [ ] 1 trial на аккаунт (повторный — ошибка)
- [ ] Trial недоступен бывшим платным подписчикам

### Auto-Conversion
- [ ] По истечении 7 дней → списание 3 900 ₽ → status = active
- [ ] Неуспешное списание → grace period (24ч) + 3 retry
- [ ] 4 неуспешных → expired
- [ ] Cancelled trial → НЕ конвертируется

### Отмена
- [ ] Отмена в любой момент через ЛК
- [ ] Доступ сохраняется до trial_ends_at
- [ ] Списания нет
- [ ] trial_used остаётся true

### Email
- [ ] T1: Welcome при активации
- [ ] T4: За 24 часа до списания (обязательно)
- [ ] T5: За 1 час до списания (обязательно)
- [ ] T6: При конвертации
- [ ] T8: При неуспешном списании
- [ ] Идемпотентность: нет дублей

### UI
- [ ] Trial CTA видим для eligible пользователей
- [ ] Trial CTA скрыт для trial_used и active подписчиков
- [ ] ЛК: прогресс-бар trial, оставшиеся дни
- [ ] ЛК: cancelled trial state
- [ ] ЛК: grace period state

### Staged Rollout
- [ ] Feature flag % rollout работает
- [ ] Consistent: один пользователь всегда видит одинаковый вариант

## Phase C: Cancellation Flow

### Flow
- [ ] 6 причин отмены
- [ ] «Нет времени» → save-offer пауза
- [ ] «Дорого» → скидка 30% на месяц (primary) + мультимесячные (secondary)
- [ ] Другие причины → прямое подтверждение
- [ ] Финальное подтверждение с датой окончания доступа

### Save-offer скидка
- [ ] Скидка 30% на следующее одно списание
- [ ] Далее — полная цена
- [ ] Лимит: 1 скидка / 6 мес (общий с win-back)
- [ ] Лимит исчерпан → только мультимесячные, без скидки

### Подтверждение
- [ ] Дата окончания доступа отображается
- [ ] Мультимесячный: доступ до конца оплаченного периода
- [ ] Месячный: доступ до конца месяца
- [ ] Кнопка «Остаться» возвращает в ЛК

---

## Phase D: Retention & Extensions

### Пауза
- [ ] Доступна активным подписчикам (не trial, не paused, не cancelled)
- [ ] 30 дней, 1 раз в 6 мес
- [ ] Read-only доступ к пройденным урокам
- [ ] Новый контент заблокирован, баннер на каждой странице
- [ ] Авто-возобновление через 30 дней + списание
- [ ] Досрочное возобновление (списание в момент resume)
- [ ] Отмена из паузы возможна
- [ ] H1, H2, H3 emails

### Win-back
- [ ] Trial: 4 письма (+1д, +7д, +14д, +30д)
- [ ] Paid: 5 писем (+1д, +7д, +14д, +30д, +60д)
- [ ] Скидки с лимитом (1 раз / 6 мес, общий)
- [ ] Без скидки если лимит исчерпан
- [ ] Скидка на 1 месяц, далее полная цена
- [ ] Оффер действует 7 дней
- [ ] Исключены: paused, active, unsubscribed

### Upgrade
- [ ] Месячный → более длинный тариф
- [ ] Отмена старого + оплата нового (atomic)
- [ ] Остаток текущего периода «дарим»

### Антифрод
- [ ] Дашборд: trial по IP, по карте, по email domain
- [ ] Alert: >5 trial/IP/24h
- [ ] Alert: 1 карта на >2 аккаунта
