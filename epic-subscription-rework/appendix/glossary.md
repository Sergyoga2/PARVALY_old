# Glossary

| Термин | Определение |
|--------|------------|
| **Trial** | Бесплатный 7-дневный пробный период с привязкой карты. По окончании — автоматическое списание 3 900 ₽/мес |
| **Auto-conversion** | Автоматический переход trial → платная подписка по истечении 7 дней |
| **Grace period** | Период (до 72 часов / 3 дня) после неуспешного списания, в течение которого доступ сохраняется. CloudPayments управляет retry автоматически |
| **Retry** | Повторная попытка списания при неуспешном платеже. CloudPayments делает 3 попытки с интервалом 1 день. После 3-й неуспешной — автоматически отменяет подписку |
| **Save-offer** | Предложение удержания в момент отмены подписки (пауза, скидка, upgrade) |
| **Win-back** | Серия email-писем для возврата ушедших пользователей, с нарастающими скидками |
| **Пауза** | Приостановка подписки на 30 дней с read-only доступом к пройденному контенту |
| **Upgrade** | Переход на более длительный тариф (например, 1 мес → 6 мес) |
| **Legacy-тариф** | Тарифный план, который больше не доступен для новых подписок, но действует для текущих подписчиков |
| **Generation** | Поколение тарифного плана: `legacy` (старые) или `new` (новые) |
| **Churn** | Отток пользователей (отмена подписки) |
| **Retention** | Удержание пользователей (% подписчиков, оставшихся за период) |
| **RPV** | Revenue per Visitor — выручка на одного посетителя. Основная метрика A/B теста |
| **MRR** | Monthly Recurring Revenue — ежемесячная рекуррентная выручка |
| **LTV** | Lifetime Value — суммарная выручка от одного подписчика за всё время |
| **ARPU** | Average Revenue Per User — средняя выручка на пользователя |
| **Feature flag** | Механизм включения/выключения функционала для % пользователей без деплоя |
| **Staged rollout** | Постепенное включение: 20% → 50% → 100% пользователей |
| **CloudPayments** | Платёжная система. API: `https://api.cloudpayments.ru`. Auth: HTTP Basic. Sandbox: test terminal ID. Rate limit: 30 запросов (prod), 5 (test) |
| **Card token** | Токенизированные данные карты (`tokenize: true` в виджете). Привязан к карте, не к подписке. Формат: строка |
| **SubscriptionId** | ID подписки в CloudPayments. Формат: `"sc_XXXXXXXXXXXX"` (строка) |
| **Webhook** | HTTP callback от CloudPayments. Типы: Pay, Fail, Recurrent, Cancel, Refund, Check, Confirm. Верификация через HMAC |
| **StartDate** | Параметр API CP: дата первого списания. Используется для trial (StartDate = now + 7 days) |
| **ДПО** | Дополнительное профессиональное образование — отдельные профессии, продаются вне подписки |
| **Оферта** | Публичный договор-оферта, описывающий условия подписки, trial и автосписания |
| **54-ФЗ** | Федеральный закон о применении ККТ. Может требовать чек при trial (0 ₽) |
| **Cancellation flow** | Многошаговый процесс отмены подписки с причинами и save-offers |
| **Cooldown** | Период ожидания между повторными использованиями функции (напр. 6 мес для скидки / паузы) |
